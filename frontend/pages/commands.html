Poll

*poll

Overview

The *poll command can be used in order to create polls. It returns an embed message with a random color. A timestamp is printed in the footer of the message. The timestamp changes accordingly to the date. If the poll was made yesterday, the timestamp would read Yesterday at <Time>. Otherwise it reads <Date>.

After the bot messages the embed poll, it reacts to the message accordingly to the options. The reactions can be used by the users to vote. The bot does not keep track of who is voting and when the vote was submitted.

Keep in mind that the initial reactions by the bot are counted in the total reactions/votes by Discord. If you want to see, how many people voted for a certain option, substract 1 from the total reactions count, that is displayed under the message.

Usage

The *poll command uses a simple pattern to recognize the question and the options. The pattern looks like this:

*poll question? option1! option2! ... option20!

The question? has to end with an ?. It can be at least 2 lines in the embed message (theoretically the question can be even longer). An error message is sent if no question is present in the command arguments:
